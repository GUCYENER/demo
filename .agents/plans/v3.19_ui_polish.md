# UI/UX Polish — v3.19.0

> v3.17.0 + v3.18.0'da eklenen Veri Kaynağı Yetkilendirme Modalı ve Sistem Özellikleri Paneli SaaS standartlarına çıkarılıyor.
> Plan ajanı tarafından üretildi, kullanıcı onayı alındı, kararlar verildi.

## Verilen Kararlar

| Konu | Karar |
|---|---|
| Toast API | A — proje standardı `window.showToast(msg, type)` |
| Tooltip | CSS-only `[data-tooltip]` helper (zero-dep) |
| Loading | Skeleton (liste/panel) + spinner (buton içi action) |
| Modal a11y | ESC + click-outside + aria-label + return-focus (focus-trap atlandı) |
| Empty state CTA | Pasif (görsel iyileştirme yeterli) |
| Card aksan renkleri | kb=mavi, db=yeşil, llm=mor |
